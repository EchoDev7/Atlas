# نصب لوگوی Atlas

## مراحل نصب لوگو:

1. فایل لوگوی PNG را با نام `atlas-logo.png` ذخیره کنید
2. آن را در مسیر زیر قرار دهید:
   ```
   /Users/majlotfi/Desktop/Atlas/frontend/assets/images/atlas-logo.png
   ```

## دستور کپی سریع:

اگر فایل لوگو را در Desktop ذخیره کرده‌اید، این دستور را اجرا کنید:

```bash
cp ~/Desktop/atlas-logo.png /Users/majlotfi/Desktop/Atlas/frontend/assets/images/atlas-logo.png
```

## تأیید نصب:

بعد از کپی کردن فایل، این دستور را اجرا کنید تا مطمئن شوید:

```bash
ls -lh /Users/majlotfi/Desktop/Atlas/frontend/assets/images/atlas-logo.png
```

## رنگ‌بندی لوگو:

لوگوی Atlas دارای گرادیانت زیبای Cyan به Teal است:
- رنگ بالا (Cyan): `#22d3ee`
- رنگ پایین (Teal): `#0891b2`
- رنگ میانی (Sky Blue): `#0ea5e9`

این رنگ‌ها در تمام صفحات پنل برای Dark/Light Mode استفاده شده‌اند.
